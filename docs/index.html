Fleet

並走する Claude Code Agent を「艦隊」として率いる。

複数の Claude Code セッションをカンバンのカードとして並べ、どのエージェントが何をしているか（稼働中・承認待ち・完了）を、ターミナルを開かずに一望する macOS アプリ。

$ launch a fleet of agents ▸
▸ ダウンロード GitHub で見る
macOS 26+ · 無料・オープンソース (MIT) · 未署名ビルド
What it does

ターミナルを開かずに、艦隊を把握する

● WORKING / ● BLOCKED

Agent 状態の自動検出

稼働中・承認待ち・完了・待機を端末出力から判定。稼働機はレーダーのブリップのように脈打つ。

$ approve? (y/n)

承認待ちの「実際の問い」

ブロック中のエージェントが何を尋ねているかをカード上に表示。点滅キャレット付き。

▸ open terminal

フル装備のターミナル

カードから全画面ターミナル（SwiftTerm）を起動。閉じてもセッションは生き続ける。

⎇ branch · #PR

cwd / ブランチ / PR

作業ディレクトリ・現在ブランチ・GitHub PR をカードに集約。ホバーで全文表示。

◇ mermaid

Markdown プレビュー

mermaid 作図とシンタックスハイライトに対応。ネットワーク不要のオフライン描画。

◐ themes · 集計

テーマ & トークン使用量

ターミナルの配色/フォント設定、Claude Code のトークン使用量を期間別に集計。

Why "Fleet"

並走する Agent は、一隻ずつではなく艦隊で動く。
Fleet はローカル完結の司令塔として、散らばったセッションを一枚のレーダーに集約する。

Install

導入

⚠️ 未署名・未 notarize ビルドのため、初回起動は Gatekeeper に弾かれます。下記いずれかで開いてください。
  1. Releases から Fleet.app.zip をダウンロードし展開
  2. Fleet.app を /Applications へ移動
  3. 初回のみ 右クリック → 開く、または検疫属性を除去: 
    xattr -dr com.apple.quarantine /Applications/Fleet.app
